build docker image from Dockerfile

docker build .


build our image using docker compose configuration

docker-compose build


use docker compose to run service from docker-compose file (app service for us)

docker-compose run <service_name>

    --> anything we add after this command will be run on our docker process



APP COMMANDS:

current directory: locally - ./app; docker /app --> WORKDIR

create project in current directory:
    docker-compose run app sh -c "django-admin.py startproject app ."

create core app:

  docker-compose run app sh -c "python manage.py startapp core"


migrate the database:

  docker-compose run app sh -c "python manage.py makemigrations && python manage.py migrate"


run tests:

  docker-compose run app sh -c "python manage.py test"
